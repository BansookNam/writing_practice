## Run locally

```bash
./start.sh
```

Then open [http://localhost:3456](http://localhost:3456) in your browser.

> Requires [Node.js](https://nodejs.org) for `npx serve`. Alternatively, use Python:
> ```bash
> python3 -m http.server 3456
> ```

---

## Deploy to GitHub Pages

1. Push to the `main` branch of your repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Save — your site will be live at:
   **https://bansooknam.github.io/writing_practice/**
